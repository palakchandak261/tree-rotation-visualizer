🌳 Tree Comparison Visualizer
BST vs AVL vs Red-Black Tree (Interactive Web Visualizer)

An interactive web-based visualization tool to compare Binary Search Tree (BST), AVL Tree, and Red-Black Tree operations in real time.

This project helps students understand:

How insertions affect tree structure

How rotations work

Why self-balancing trees are important

Height differences between BST, AVL, and RB trees

✨ Features
🌲 1. Three Trees Side-by-Side

Binary Search Tree (BST)

AVL Tree

Red-Black Tree

All trees update simultaneously when inserting a value.

🔁 2. Independent Rotation Tracking

AVL rotations shown inside AVL card

Red-Black rotations shown inside RB card

📊 3. Live Statistics Panel

For each tree:

Height

Total Rotations

Real-time comparison

📉 4. Visual Graphs

Height comparison graph
Bar Graph


📏 5. Level Guides (Visual Understanding Boost 🚀)

Dashed horizontal lines for each level

Subtle light grey styling

Low opacity level labels (L0, L1, L2...)

Improves structural clarity

Helps understand height visually

🎨 6. Clean Modern UI

Card-based layout

Responsive design

Soft shadows

Smooth animations

Tree cards separated for clarity

🧠 Concepts Demonstrated
Binary Search Tree (BST)

No balancing

Can become skewed

Height may grow to O(n)

AVL Tree

Strictly height balanced

Performs rotations:

LL

RR

LR

RL

Height remains O(log n)

Red-Black Tree

Color-based balancing

Fewer rotations than AVL

Maintains near-balanced height

Rules enforced:

Root is black

No two red nodes adjacent

Equal black height paths
