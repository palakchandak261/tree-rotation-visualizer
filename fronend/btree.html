B-Tree Complete Guide

What is a B-Tree?

A B-Tree is a self-balancing multi-way search tree used in databases and file systems.

  • All leaves are at same level
  • Nodes contain multiple keys
  • Keys are sorted

Insertion & Split

  • Insert into correct leaf
  • If overflow → split node
  • Middle key moves to parent
  • Root split increases height

Deletion

  • Remove from leaf
  • Borrow from sibling if needed
  • Merge if underflow

Time Complexity

  • Search → O(log n)
  • Insert → O(log n)
  • Delete → O(log n)

Real World Usage

  • Database Indexing (MySQL, PostgreSQL)
  • File Systems (NTFS, EXT4)
  • Large storage systems
